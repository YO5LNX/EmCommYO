# Comunitatea EmCommYO

Folosim [wiki-ul](./wiki) acestui repo pentru a organiza informații relaționate cu activitățile din domeniul comunicațiilor de urgență (EmComm) în România, adunate de comunitatea EmCommYO.

În acest repo o să se regăsească anumite fișiere de interes create de comunitatea noastră. În acest moment avem o [librărie de conținut pentru agregatoare tip RSS/Atom](./EmCommYO.opml) în format OPML, care este suportat pentru import în majoritatea acestor tip de aplicații.
